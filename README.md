E-Arsip Digital - Integrated Archiving System

E-Arsip Digital adalah aplikasi manajemen kearsipan modern berbasis web yang dirancang untuk membantu instansi dalam digitalisasi, pengorganisasian, dan pengamanan dokumen secara terpusat. Aplikasi ini dikembangkan dengan fokus pada kemudahan penggunaan (user experience) dan keamanan data tingkat tinggi.

Fitur Unggulan (Enterprise Ready)
Aplikasi ini dilengkapi dengan berbagai fitur kelas profesional:

- Role-Based Access Control (RBAC): Pembagian hak akses dan wewenang yang jelas antara *Admin* (kontrol penuh sistem) dan *Pegawai* (akses terbatas sesuai tupoksi).
- Dynamic Branding: Pengaturan logo dan nama instansi yang dinamis. Administrator dapat mengubah identitas aplikasi secara langsung melalui panel pengaturan UI tanpa perlu menyentuh kode program.
- Pencarian Global (Smart Search): Mesin pencari terintegrasi untuk menemukan dokumen spesifik yang tersebar di seluruh folder klasifikasi hanya dengan menggunakan satu kata kunci.
- JRA (Jadwal Retensi Arsip) Otomatis: Perhitungan masa aktif dan inaktif dokumen secara otomatis berdasarkan tahun pembuatan dan kebijakan retensi.
- Audit Trail (Activity Log): Pencatatan otomatis untuk setiap aktivitas pengguna (login, tambah, ubah, hapus, unduh) guna memastikan transparansi dan keamanan sistem.
- Data Management: Mendukung ekspor laporan kearsipan ke format `.xlsx` / `.pdf` dan impor data massal dari file Excel.
- Safety Net (Soft Delete): Fitur pengamanan "Tong Sampah" untuk mencegah kehilangan data atau file fisik secara permanen akibat ketidaksengajaan pengguna.
- One-Click Backup: Fitur pencadangan (backup) *database* langsung ke dalam format `.sql` melalui antarmuka web, tanpa perlu membuka panel server.

Teknologi yang Digunakan

* Backend Framework: Laravel (PHP)
* Database: MySQL / MariaDB (via HeidiSQL/Laragon)
* Frontend & UI: HTML5, CSS3, JavaScript, Bootstrap 5
* Iconography: FontAwesome 6
* Library Pendukung:
    * `maatwebsite/excel` (Manajemen data Spreadsheet)
    * `barryvdh/laravel-dompdf` (Generasi dokumen PDF)
    * `sweetalert2` (Interactive Pop-up UI)
