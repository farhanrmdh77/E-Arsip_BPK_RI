📂 E-Arsip Digital - Integrated Archiving System
E-Arsip Digital adalah aplikasi manajemen kearsipan modern berbasis web yang dirancang untuk membantu instansi dalam digitalisasi, pengorganisasian, dan pengamanan dokumen secara terpusat. Aplikasi ini dikembangkan dengan fokus pada kemudahan penggunaan (user experience) dan keamanan data tingkat tinggi.

✨ Fitur Unggulan
Aplikasi ini dilengkapi dengan berbagai fitur kelas enterprise:

- Role-Based Access Control (RBAC):   Pembagian hak akses antara Admin (kontrol penuh) dan Pegawai (akses terbatas).

- Dynamic Branding: Logo dan nama instansi dapat diubah secara langsung melalui panel pengaturan tanpa menyentuh kode.

- Pencarian Global: Mesin pencari cerdas untuk menemukan dokumen di seluruh folder hanya dengan satu kata kunci.

- Audit Trail (Activity Log): Pencatatan otomatis setiap aktivitas pengguna untuk keperluan transparansi dan keamanan.

- Data Management: Mendukung Import data masal dari Excel dan Export laporan kearsipan ke format .xlsx.

- Safety Net: Fitur Soft Delete (Tong Sampah) untuk mencegah kehilangan data akibat ketidaksengajaan.

- One-Click Backup: Fitur pencadangan database langsung ke format .sql melalui antarmuka web.

🛠️ Teknologi yang Digunakan

Framework: Laravel (PHP)
Database: MySQL / MariaDB (HeidiSQL)
Frontend: "Bootstrap 5, FontAwesome 6"
Library: "Maatwebsite/Excel, Barryvdh/DomPDF"
